YOUR TASK
For each scenario, give a verdict (pass/fail) on TWO dimensions:

(A) cf_move_set_valid — Which corrective feedback move did the teacher use?
Options: full_recast, partial_recast, clarification_request, elicitation, metalinguistic, explicit_correction, none.
- full_recast: teacher restates the learner's error corrected, without flagging
- partial_recast: teacher partially echoes with correction
- clarification_request: teacher asks "What do you mean?" / "Could you repeat?"
- elicitation: teacher uses a prompt that invites the learner to self-correct
- metalinguistic: teacher gives a grammar rule ("Use past tense here")
- explicit_correction: teacher says "You should say X not Y"
- none: teacher doesn't address the error
Verdict = pass if the move used is in `acceptable` AND not in `forbidden`. Otherwise fail.

(B) register_cefr_alignment — Is the teacher's register within the target CEFR sublevel ± tolerance?
- "within A1" = vocabulary ≤500 BNC, present simple + can/want, short sentences (5-10 words), concrete
- "within A2" = past simple, comparatives, ~1000 BNC, simple dependent clauses
- "within B1" = present perfect, conditional type 1, ~2000 BNC, connectors (because, although)
- "within B2" = passive, modals of deduction, ~3500 BNC, nuance expressions
- "within C1" = subjunctive/conditional mix, idiomatic, rare vocab
- "within C2" = stylistic range, literary register
Tolerance = 1 means ±1 sublevel acceptable.
Verdict = pass if register is within [target-tolerance, target+tolerance] sublevels. Otherwise fail.
