# Pedagogy — AcademIA

## Teaching approach: TTT (Test-Teach-Test)

Adaptive cycle per concept:
1. Test: assess current level
2. Teach: mini-lesson on gaps
3. Test: verify improvement

Deterministic concept transitions (code_turn_check node in Teacher chatflow).
Focus concept changes every N turns based on performance.

## Curriculum strategy

- No RAG on commercial textbooks
- Curriculum generated by Claude Sonnet (Sinse Pro subscription)
- Stored in PostgreSQL
- Taxonomy defined per domain before generation

## English taxonomy (A1→C2)

- A1 Survival: correct phrases, greetings, simple questions
- A2 Daily: present simple/continuous, past, basic modals
- B1 Autonomy: present perfect, passive, conditionals 1&2, phrasal verbs
- B2 Fluency: conditional 3, mixed conditionals, deduction modals, registers
- C1 Mastery: idioms, nominalization, pragmatics, dialects
- C2 Excellence: multiple registers, rhetoric, indistinguishable from native

## Assessment

- Confidence scores per concept: 0-100 in PostgreSQL (scores_confiance JSONB)
- Promotion guard: days_seen >= 2
- Tracking: {score, last_seen, first_seen, days_seen}
- Mock exam: 10 questions via gpt-4o-mini, format feedback
