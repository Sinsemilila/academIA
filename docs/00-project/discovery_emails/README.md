4 emails non-contractuels pour débloquer ou préciser l'accès à des corpus
learner annotés par langue. Tous non-bloquants : les stratégies Wave 1-4
fonctionnent sans ces réponses, mais une réponse positive débloque du gain
qualitatif.

## Protocole d'envoi

1. Expéditeur : sinseproduction@gmail.com
2. Ton : chercheur indépendant, projet éducatif familial, open-source-friendly
3. Langue : français si interlocuteur FR, sinon anglais
4. Pas de pitch business, pas d'urgence
5. Remerciements pour leur travail en ouverture
