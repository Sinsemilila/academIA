## Violations détaillées

### V1. Link `text-text-muted` insufficient contrast (FIXED)

- **Element** : `<a class="text-xs text-text-muted hover:text-text-secondary">Mentions légales</a>` (footer Login + autres routes)
- **Contrast** : 2.43:1 (foreground #9ca3af on background #fafafa) — target 4.5:1
- **Cause** : `--color-text-muted` light theme = `oklch(0.714 0.019 261.325)` → trop clair
- **Fix shipped S57** : bump to `oklch(0.530 0.019 261.325)` → contrast ~4.7:1 ✅
- **Side effect** : tous contexts text-muted en light theme (Header XP, KeyboardShortcuts kbd, ProgressionGraph axes, etc.) gagnent en lisibilité. No regression dark theme (token séparé).
- **Validation** : requires frontend rebuild + container restart pour propager (production build, pas dev server)

### V2. Button `bg-teacher` insufficient contrast (DEFERRED — design decision)

- **Element** : `<button class="bg-teacher text-white">Entrer</button>` (Login submit + autres CTAs)
- **Contrast** : 3.67:1 (foreground #ffffff on background #3b82f6) — target 4.5:1
- **Cause** : `--color-teacher: oklch(0.623 0.188 259.815)` (= #3b82f6, ex-Tailwind blue-500) — agent accent **shared cross-themes par convention B1** (`docs/99-runbooks/b1-design-tokens.md` : "perception-pinned hue+chroma")
- **Status** : DEFERRED — décision design Sinse needed
- **Options** :
  - **A. Darken teacher cross-theme** : oklch(0.55 0.18 259.815) ≈ #2563eb (Tailwind blue-600), contrast ~5:1 ✅ MAIS impact branding all 7 agent accents (maestro/sensei/lehrer/etc. à re-checker pour cohérence + maintenir le balance perceptuel inter-agents)
  - **B. Add `bg-teacher-strong` token** : keep brand color `--color-teacher`, ajout variant darker pour buttons specifically. Plus de tokens à maintenir.
  - **C. Keep current, accept WCAG AA fail** : marketing/branding consistency over a11y AA on agent CTAs. Add `aria-label` for clarity. Quasi-impossible à justifier — le button text "Entrer" est purement fonctionnel, pas branding.
  - **D. Switch text white → black on bg-teacher** : breaks visual identity, bad option.
- **Recommandation Claude** : **Option A** (darken cross-theme + re-check 6 autres accents). Maintient cohérence design system + WCAG. Need pedagogy review pour confirmer perception inter-agents préservée.
